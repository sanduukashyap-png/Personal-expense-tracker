# Personal-expense-tracker
A Python-based Personal Expense Tracker that helps users record income and expenses, view financial summaries, and save data to a CSV file using a simple command-line interface.
Personal Expense Tracker
Overview
The Personal Expense Tracker is a simple Python application that helps users manage their daily income and expenses. It allows users to add income, record expenses, view a financial summary, and save the data to a CSV file.
Features
Add income
Add expenses
View total income, total expenses, and balance
Save records to a CSV file
Easy-to-use menu-driven interface
Technologies Used
Python 3
CSV Module
How to Run
Install Python 3.
Save the code as personal_expense_tracker.py.
Open the terminal in the project folder.
Run:
python personal_expense_tracker.py
